Option A — Floating buttons only
+Maximum map real estate.
+Immersive, map-first feel.
−No brand presence in core experience.
−Controls are less discoverable — no container to hint they're tappable.
−Nowhere for future elements (fuel chip, search).
−Looks unfinished on first open.
Option B — Minimal top bar CHOSEN
+Wordmark on every open — builds brand recall with early adopters.
+Bell + hamburger in a defined chrome area — instantly discoverable.
+Loading overlay stays below bar — chrome is always accessible.
+Natural home for fuel chip (Story 2.4) when it arrives.
−~44dp of vertical map lost — acceptable on modern screen sizes.

Bar is kept deliberately minimal: white bg, 1px shadow, no border, bare icon buttons. The map is still the hero.
